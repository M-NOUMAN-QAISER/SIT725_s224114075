## How the Application Works

1. The webpage displays a heading **"SIT 725"**.
2. A button labelled **"Change Text"** is displayed.
3. When the button is clicked:
   - JavaScript selects a random text from an array.
   - The heading text is updated dynamically.

This demonstrates basic **DOM manipulation in JavaScript**.
